# BPSI
EN:
Batch processing of satellite images.
To work, you need the GDAL / OGR library.
All output files are placed in the "out" folder.
For work, satellite images of Landsat 4, 5, 7, 8 and Sentinel 2 are used.

RU:
Пакетная обработка спутниковых снимков.
Для работы необходима библиотека GDAL/OGR.
Все выходные файлы помещаются в папку "out".
Для работы используются спутниковые снимки Landsat 4, 5, 7, 8 и Sentinel 2.
